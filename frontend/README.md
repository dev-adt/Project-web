# NextJS Frontend Web Application

The frontend renders the public landing page components dynamically alongside the admin panel canvas editor, section libraries, media grids, and settings.

## Core Stack

* **NextJS 15 & React 19**: Responsive Server Side Rendering (SSR) routing.
* **TailwindCSS**: Premium custom stylesheet layout tokens.
* **Zustand**: Fast and decoupled client state stores.
* **Lucide Icons**: Harmonious icon library.

---

## Local Development

Install dependencies:
```bash
npm install
```

Start the application:
```bash
npm run dev
```

Application is active on: [http://localhost:3000](http://localhost:3000).
All backend API routes are mapped via NextJS proxies and reverse-proxy gateways to the Nginx portal.
